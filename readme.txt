== Description ==
This plugin adds a lightbox to the native galleries of WordPress articles.

= Features Include =
* Carousel lightbox multi galleries.
* Choice of the number of columns.
* Loading images with lazy Load.

== Configuration ==

1. Activate the plugin through the "Plugins" menu in WordPress.
2. Go to the plugin setting page, and configure to your wishes.
3. Create a new article and add a wordpress gallery and validate.
4. You can go see your article, the gallery should be present.
